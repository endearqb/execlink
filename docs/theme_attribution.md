# Terminal Theme Attribution

ExecLink includes built-in terminal color mappings inspired by open-source VS Code themes.

## Included theme IDs

- `vscode-dark-plus`
- `vscode-light-plus`
- `monokai`
- `monokai-light`
- `dracula`
- `one-light`
- `github-dark`
- `github-light`
- `nord`
- `solarized-light`
- `gruvbox-dark`
- `gruvbox-light`

## Upstream references

- VS Code default themes: <https://github.com/microsoft/vscode/tree/main/extensions/theme-defaults/themes>
- Dracula theme: <https://github.com/dracula/visual-studio-code>
- One Light theme family: <https://github.com/akamud/vscode-theme-onelight>
- GitHub theme: <https://github.com/primer/github-vscode-theme>
- Nord theme: <https://github.com/arcticicestudio/nord-visual-studio-code>
- Solarized theme family: <https://github.com/altercation/vim-colors-solarized>
- Gruvbox theme family: <https://github.com/morhetz/gruvbox-contrib>

These mappings are reduced terminal-oriented palettes and not full editor token themes.
